# crosslearn

`crosslearn` provides extractor-first reinforcement learning utilities for Gymnasium,
Stable-Baselines3, and Chronos-backed time-series features.

## Installation

```bash
pip install crosslearn
pip install "crosslearn[atari]"
pip install "crosslearn[chronos]"
```

## Highlights

- Native `REINFORCE` agent utilities.
- Atari preprocessing and feature extractors.
- Chronos-based rolling-window embedders and extractors.
- Callback, logging, and vector-environment helpers.
